# Bezpieczeństwo

## Zgłaszanie podatności

Jeśli znajdziesz lukę bezpieczeństwa związaną z tym repozytorium lub rozszerzeniem:

1. **Nie** otwieraj publicznego issue z exploitacją — dzięki temu inni użytkownicy nie są narażeni przed poprawką.
2. Zgłoś problem **prywatnie** przez [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) dla tego repozytorium (po jego opublikowaniu), albo skontaktuj się z maintainerem przez e-mail (dodaj adres w README, jeśli go używasz).

Opisz: wersję rozszerzenia / commit, przeglądarkę, kroki odtworzenia i potencjalny wpływ.

## Zakres

Dotyczy kodu w tym repozytorium i paczki publikowanej w Chrome Web Store. Problemy wyłącznie po stronie serwisów Allegro lub OLX zgłaszaj odpowiednim kanałom tych serwisów.
